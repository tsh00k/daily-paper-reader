## 摘要
本文研究多目标模仿学习：在多目标马尔可夫决策过程（MOMDP）中，给定多个帕累托最优专家的示范，恢复位于帕累托前沿的策略的问题。标准模仿方法对此情况准备不足，因为天真地聚合冲突的专家轨迹可能导致被支配的策略。为了解决这个问题，我们引入了多输出增强行为克隆（MA-BC），一种算法，它系统地划分分歧的专家数据，同时合并未观察到行为冲突的状态-动作对。理论上，我们证明MA-BC以比任何独立考虑每个专家数据集的学习者更快的统计速率收敛到帕累托最优策略。此外，我们为多目标模仿学习建立了一个新的下界，证明MA-BC是极小化最优的。最后，我们在多样的离散环境中实证验证了我们的算法，并在理论洞察的指导下，将MA-BC扩展到连续线性二次型调节器（LQR）控制任务并进行了评估。

## Abstract
This work investigates multi-objective imitation learning: the problem of recovering policies that lie on the Pareto front given demonstrations from multiple Pareto-optimal experts in a Multi-Objective Markov Decision Process (MOMDP). Standard imitation approaches are ill-equipped for this regime, as naively aggregating conflicting expert trajectories can result in dominated policies. To address this, we introduce Multi-Output Augmented Behavioral Cloning (MA-BC), an algorithm that systematically partitions divergent expert data while pooling state-action pairs where no behavior conflict is observed. Theoretically, we prove that MA-BC converges to Pareto-optimal policies at a faster statistical rate than any learner that considers each expert dataset independently. Furthermore, we establish a novel lower bound for multi-objective imitation learning, demonstrating that MA-BC is minimax optimal. Finally, we empirically validate our algorithm across diverse discrete environments and, guided by our theoretical insights, extend and evaluate MA-BC on a continuous Linear Quadratic Regulator (LQR) control task.

### 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

- **核心思想**：利用帕累托最优专家在大多数状态上行为**一致**（“共同状态”）这一结构特性，将数据划分为**冲突数据**和**共同数据**，然后将共同数据共享给所有专家，仅对每个专家的冲突数据单独处理，从而在保证帕累托最优性的同时提升样本效率。  
- **关键技术细节与算法流程（MA-BC）**：
    1. **识别分歧状态**：扫描所有专家数据集，若同一状态出现了不同动作，则将该状态标记为分歧状态，其数据归入 `Ddiv`。
    2. **定义共同状态**：所有未被标记为分歧状态的数据构成 `Dcommon`（即所有专家在该状态下动作一致）。
    3. **训练每个专家策略**：对第 ℓ 个专家，使用 `Dcommon ∪ Dℓ`（即共同数据加上该专家自身的数据）进行行为克隆（BC），得到策略 `π̂ℓ`。
    4. **输出**：`{π̂ℓ}_{ℓ=1}^{L}`。
- **理论保证**：
    - **定理3.1**：MA-BC 输出的每个策略是 εℓ-帕累托前沿策略，其中 εℓ 由两项组成：  
      `ν_{πℓ}/ν_{πdata} ∞ * |X_common| / (e(1-γ)² N) + |X \ X_common| / (e(1-γ)² Nℓ)`。  
      - 第一项表明：共同状态越多、联合数据量 N 越大，公共学习的速率越快（受集中率系数 `ν_{πℓ}/ν_{πdata}` 调控）。  
      - 第二项表明：分歧状态 K 仅要求每个专家自身的数据量 Nℓ 与之线性相关，而非整个状态空间。
    - **定理3.2（下界）**：任何离线多输出模仿算法都无法避免上述两项中的较大者，即 MA-BC 是**极小化最优**的。
- **连续扩展**：对连续状态-动作空间，通过设置空间容忍度 δ，将状态距离 ≤ δ 且动作距离 ≤ L_{πℓ}·δ 的异专家数据视为“兼容”并共享；存在偏差-数据权衡（δ 越大共享越多，但引入的偏置也越大）。

### 6. 论文的主要结论与发现

1. **Naive BC（合并数据）必然失败**：即使数据无限，也会学习到被支配策略（Failure II），理论证明存在恒定次优间隙（定理2.2）。
2. **Isolated BC（独立学习）样本效率低**：要求每个专家的数据量与状态空间大小线性相关（Failure I）。
3. **MA-BC 同时避免两种失败**：通过状态级的数据划分和共享，以更快的统计速率收敛到帕累托前沿（定理3.1）。
4. **MA-BC 是极小化最优的**：建立了匹配的下界（定理3.2）。
5. **连续扩展存在偏差-数据权衡**：容忍度 δ 越大，共享数据越多但偏置也越大；最优 δ 随已有数据量增加而减小（更多数据时应优先减少偏置）。
6. **集中率 `Cπdata` 是关键因素**：当各专家访问同一状态的概率相近时，MA-BC 优势最大。
