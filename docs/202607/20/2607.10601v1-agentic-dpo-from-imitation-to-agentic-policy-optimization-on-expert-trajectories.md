## 摘要
大型语言模型（LLM）智能体通常通过监督微调（SFT）从专家轨迹中训练，这种方法将多轮智能体行为视为普通文本模仿。这种方案简单且成本低，但仅学习模仿专家动作序列，而非训练智能体在每个状态下针对可能的错误选择正确动作。现有缓解该问题的方法包括偏好学习或强化学习，但通常需要高成本的环境交互和奖励模型。我们提出Agentic-DPO，一种轻量级离线智能体策略优化方法，将专家轨迹转化为状态条件偏好监督。在每个专家动作状态，Agentic-DPO从当前状态采样一步动作，将合理的错误动作作为负例，并使用DPO风格的偏好目标将其与专家动作对比。为避免偏好学习中混合策略和模式，我们引入策略保持增强（PPA），即在保持专家策略固定的同时，在多种模式下渲染相同的潜在轨迹。Agentic-DPO无需在线环境交互、奖励模型或完整轨迹的学生探索。我们在StableToolBench、tau-bench零售和Mind2Web上进行实验，Agentic-DPO在不同模型规模上持续提升智能体性能，超越模仿基线。特别是，对于9B模型，它将tau-bench准确率从21.7%（SFT）提升至41.4%，在相同主干网络下仅需单步交互且无需梯度更新期间的环境交互，即可匹配在线GRPO。结果表明，当从演示转换为状态级动作偏好时，专家轨迹可支持低成本智能体策略优化。Agentic-DPO代码已开源至https://github.com/Schuture/Agentic-DPO。

## Abstract
Large Language Model (LLM) agents are commonly trained from expert trajectories using supervised fine-tuning (SFT), which treats multi-turn agent behavior as ordinary text imitation. This recipe is simple and low-cost, but it only learns to imitate the sequence of expert actions, rather than training the agent to choose the right action against plausible mistakes at each state. Existing methods to mitigate this problem include preference learning or reinforcement learning, but they usually need high-cost environment rollouts and reward models. We propose Agentic-DPO, a lightweight offline agent policy optimization method that turns expert trajectories into state-conditioned preference supervision. At each expert action state, Agentic-DPO samples a one-step action from the current state, treats plausible wrong actions as negatives, and contrasts them with the expert action using a DPO-style preference objective. To avoid mixing both policy and schema in preference learning, we introduce Policy-Preserving Augmentation (PPA), which renders the same latent trajectory under multiple schemas while keeping the expert policy fixed. Agentic-DPO requires no online environment rollout, reward model, or full-trajectory student exploration. We conduct experiments across StableToolBench, tau-bench retail, and Mind2Web, where Agentic-DPO consistently improves agents at different model scales beyond imitation. In particular, it raises tau-bench accuracy from 21.7% (SFT) to 41.4% for a 9B model, matching online GRPO under the same backbone with only step-level rollouts and without environment interaction during gradient steps. The results suggest that expert trajectories can support low-cost agentic policy optimization when converted from demonstrations into state-level action preferences. Code for Agentic-DPO is released at https://github.com/Schuture/Agentic-DPO.

- **论文目标**：提出一种**轻量级、离线**的智能体策略优化方法，仅利用已有的专家轨迹，通过构造状态级的偏好对来训练智能体，无需环境交互、奖励模型或完整轨迹探索，从而在低成本下实现超越模仿的性能。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
将专家轨迹中的**每一步专家动作**视为该状态下的“首选动作”，并从当前学生策略在该状态下**一步采样**得到的动作中选出最可能的错误动作作为“负例”，然后使用 DPO 风格的偏好损失拉大两者的概率差。这样，学生不仅模仿专家，还学会主动规避自己可能犯的错误。

### 关键技术细节

1. **State-Conditioned Action Preferences**：
   - 对于专家轨迹中的每个状态-专家动作对 \((s_t, u_t^+)\)，在当前学生策略 \(\pi_{\theta_{\text{student}}}\) 下采样 \(K=4\) 个候选动作，过滤掉无法解析或与专家动作等价的候选，从中选择**学生 log 概率最高**的候选作为负例 \(u_t^-\)（“hardest-different-action”规则）。
   - 构建偏好三元组 \((s_t, u_t^+, u_t^-)\)。

2. **Agentic-DPO Loss**：
   - 使用长度归一化的 DPO 损失：
     \[
     \mathcal{L}_{\text{ADPO}} = -\mathbb{E}_{(s, u^+, u^-), \phi}\left[ \log \sigma\left( \beta_{\text{eff}} \cdot \left( \log \frac{\pi_\theta(a^{+, \phi}|s^\phi)}{\pi_\theta(a^{-, \phi}|s^\phi)} - \log \frac{\pi_{\text{ref}}(a^{+, \phi}|s^\phi)}{\pi_{\text{ref}}(a^{-, \phi}|s^\phi)} \right) \right) \right]
     \]
     其中 \(\beta_{\text{eff}} = \beta / \max(|a^+|, |a^-|)^\alpha\)（\(\alpha=0.5\)）用于平衡不同长度动作的影响。

3. **稳定训练的两项设计**：
   - **SFT Anchor（锚点）**：在 Agentic-DPO 损失中额外添加 SFT 损失项 \(\lambda \mathcal{L}_{\text{SFT}}\)（\(\lambda=0.5\)），保持专家动作的概率不因偏好对比而被过度压低，同时保持输出格式的有效性。理论证明（Proposition 1）在初始化时 SFT 梯度与 DPO 梯度方向一致，不矛盾。
   - **Policy-Preserving Augmentation (PPA，策略保持增强)**：在同一潜在决策下，通过多种“模式视图（schema）”渲染状态和动作，包括：
     - 动作渲染改写（如 ReAct 格式 vs JSON 格式、工具重命名）
     - 状态上下文改写（如注入历史错误及恢复场景）
     这使得 DPO 对比主要依赖潜在策略差异而非表面格式差异，平均掉模式相关的梯度。

4. **训练流程（Algorithm 1）**：
   - 先用 SFT 在专家轨迹上预热（warm-up），冻结参考策略 \(\pi_{\text{ref}}\)。
   - 多轮“负例刷新（refresh rounds）”：每轮用当前学生策略对每个专家状态采样负例，构建偏好缓冲区，然后进行若干步梯度更新（使用 \( \mathcal{L}_{\text{train}} = \mathcal{L}_{\text{ADPO}} + \lambda \mathcal{L}_{\text{SFT}} \)）。
   - 重复多轮（实验中使用 \(R=5\) 轮）。

## 6. 论文的主要结论与发现

- Agentic-DPO 在所有 benchmark 和所有模型规模上一致优于 SFT 及各种强基线（ETO、GRPO、DFT），提升幅度显著（如 StableToolBench 2B 上 canonical 从 57.1%→90.9%）。
- 在 τ‑bench 9B 上，Agentic-DPO 达到 41.4%，**与在线 GRPO（40.0%）持平**，但计算成本仅为 GRPO 的约 1/8。
- **关键发现**：只需在专家状态上采样一步负例，无需完整轨迹或环境交互，即可获得有效的策略对比信号；PPA 和 SFT 锚点是稳定训练的关键。
- 数据效率高：仅用 25% 专家轨迹即可达到 PPA+SFT 使用 100% 数据的性能。

## 7. 优点

- **低成本高性能**：完全离线，不依赖环境交互、奖励模型或人类反馈，训练开销仅略高于 SFT，性能却接近甚至超过在线 RL。
- **创新设计**：
  - 将 DPO 从轨迹级/任务级压缩到**单步动作级**，且负例来自当前学生一步采样，避免了全轨迹探索。
  - PPA 有效解耦了“策略决策”与“表面格式”，使偏好学习聚焦于真实决策差异。
  - SFT 锚点理论证明与 DPO 梯度一致，稳定训练。
- **实验覆盖全面**：多 benchmark、多模型尺寸、多基线、多消融，验证了方法的鲁棒性和通用性。
- **开源代码**：已公开，便于复现和扩展。

## 8. 不足与局限

- **仍为离线方法**：只能利用专家轨迹中已出现的状态，无法探索学生自身长程行为所产生的新状态。这是与在线 RL 的根本区别。
- **对专家轨迹质量依赖**：若专家轨迹覆盖的关键决策状态不足，或存在错误，Agentic-DPO 可能无法获得足够好的偏好信号。
- **适用场景限制**：论文指出，对于需要远超出专家数据探索的任务，可能仍需在线环境交互。
- **训练时长未明确**：只给了相对成本比，未提供绝对训练时间，可能影响实际部署判断。
- **部分 benchmark 缺乏 GRPO 对比**（如 Mind2Web），但原因合理（缺乏步级奖励）；不过读者可能期望更多 RL 对比。
