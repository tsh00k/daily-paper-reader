## 摘要
模仿学习能够利用专家演示在具有潜在奖励信号的未知环境中学习策略，但当模仿者与专家的观察不匹配且专家演示中存在未观测的混杂因素时，模仿学习会陷入困境。通过顺序π-后门准则识别适当的调整集，因果模仿学习（CIL）提供了一个从混杂数据中近似专家策略的框架。然而，现有的CIL方法，即因果行为克隆（Causal BC）和因果生成对抗模仿学习（Causal GAIL），是为短时域、低维设置设计的。当应用于具有长时域和高维状态-动作空间的连续控制任务时，这些方法表现不佳：Causal BC遭受复合误差，Causal GAIL不稳定且样本效率低，而顺序π-后门调整变得不切实际。我们提出了因果软Q模仿学习（Causal SQIL）和因果逆软Q学习（Causal IQ-Learn），这是两种离策略因果模仿学习算法，它们将因果调整框架与最先进的逆强化学习目标相结合。这两种算法都基于对顺序π-后门准则的高效近似所产生的因果调整状态表示进行操作，利用连续控制环境的因果结构将全时域调整简化为固定大小的滑动窗口。我们在一个受混杂因素干扰的环境套件中评估所有方法，发现Causal SQIL和Causal IQ-Learn在长时域任务上显著优于先前的CIL算法，有时甚至超越专家，而所有非因果的模仿方法均无法学习到有意义的行为。

## Abstract
Imitation learning enables learning a policy in an unknown environment with a latent reward signal using expert demonstrations, but it struggles when the imitator's and expert's observations are mismatched and unobserved confounders are present in expert demonstrations. By identifying appropriate adjustment sets via the sequential $π$-backdoor criterion, causal imitation learning (CIL) provides a framework for approximating the expert's policy from confounded data. However, existing CIL methods, Causal Behavioral Cloning (Causal BC) and Causal Generative Adversarial Imitation Learning (Causal GAIL), are designed for short-horizon, low-dimensional settings. When applied to continuous control tasks with long horizons and high-dimensional state-action spaces, these methods exhibit poor performance: Causal BC suffers from compounding errors, Causal GAIL is unstable and sample-inefficient, and sequential $π$-backdoor adjustment becomes impractical. We introduce Causal Soft Q Imitation Learning (SQIL) and Causal Inverse soft-Q Learning (IQ-Learn), two off-policy causal imitation learning algorithms that combine the causal adjustment framework with state-of-the-art inverse reinforcement learning objectives. Both algorithms operate on causally-adjusted state representations produced by an efficient approximation of the sequential $π$-backdoor criterion, exploiting the causal structure of continuous control environments to reduce the full-horizon adjustment to a fixed-size sliding window. We evaluate all methods in a suite of confounded environments and find that Causal SQIL and Causal IQ-Learn substantially outperform prior CIL algorithms on long-horizon tasks, sometimes surpassing the expert, whereas all causally unaware imitation methods fail to learn meaningful behavior.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：模仿学习（IL）通过专家演示学习策略，但通常假设专家和模仿者观测完全一致（无未观测混杂，NUC）。实际场景中，专家可能拥有额外传感器或环境存在潜在外力（如风、地震），导致未观测混杂变量同时影响状态转移、专家动作和奖励，使观测数据中的相关关系偏离因果机制。
- **现有局限**：因果模仿学习（CIL）利用因果图（如顺序π-后门准则）确定安全的调整集以消除混杂偏差，但已有方法（Causal BC、Causal GAIL）仅适用于短时域、低维任务。在长时域（H≥1000）和高维连续控制（如MuJoCo蚂蚁、人形机器人）中，Causal BC因复合误差崩溃，Causal GAIL因依赖在策略优化和不稳定性而失败，且全时域调整集计算不可行。
- **本文目标**：提出**Causal SQIL**和**Causal IQ-Learn**两种离策略因果模仿算法，将因果调整框架与软Q学习结合，并引入**窗口化顺序π-后门调整**，使得因果模仿在长时域高维连续控制中可扩展且有效。

## 2. 论文提出的方法论

### 核心思想
- 基于已知因果图，利用**顺序π-后门准则**识别每个时间步应条件化的观测变量子集（排除混杂代理变量，如受风影响的罗盘）。
- 提出**滑动窗口近似**（Algorithm 1），假设混杂影响的时域有界（**k-有界时间齐次混杂**，Assumption 1），将全时域调整集计算简化为在固定长度k的代理图上计算一次，然后应用于全时域（Theorem 1证明正确性）。这样避免了随H线性增长的维度和计算开销。
- 调整后的状态表示`z_t`通过编码滑动窗口内相关变量的值获得。

### 关键技术细节
- **Causal SQIL**：在调整后的状态`z_t`上运行SQIL（Soft Q Imitation Learning）。SQIL将专家转移标记奖励1，策略转移标记奖励0，然后训练SAC（软演员-评论家）代理。评论家最小化软贝尔曼残差，演员最大化熵正则化目标。公式(1)(2)与标准SQIL相同，但输入为`(z,x)`。
- **Causal IQ-Learn**：在调整后的状态`z_t`上运行IQ-Learn（Inverse soft-Q Learning）。IQ-Learn学习一个Q函数，其隐式奖励与专家行为一致。评论家通过卡方散度损失在专家数据上约束软贝尔曼方程，并加上策略一致性正则化。演员目标与SQIL相同。公式(3)(4)。
- 两个算法均使用**双Q网络、软目标更新、自动熵调整**等标准SAC技巧。
- **窗口化调整算法流程**（Algorithm 1）：
  1. 构建短时域代理图`G_k`（时域`k+1`）。
  2. 运行`FIND_OX`获得最大可调整观测集`O_X`。
  3. 根据马尔可夫边界及边界动作构建每个时间步的调整集`Z_t^k`。
  4. 通过时间齐次性，将`Z_t^k`剪裁为固定宽的窗口`Z_t^H`。
  5. 返回滑动窗口规范`S`，用于编码`z_t`。

## 3. 实验设计

  - 非因果：标准**BC**, **GAIL**, **SQIL**, **IQ-Learn**（条件于全部观测`V_O`）。

### 主要实验结果
- 所有非因果方法在所有任务上**几乎完全失败**（成功率0%或近0%），验证了混杂导致的学习偏差。

## 6. 论文的主要结论与发现

1. **因果调整是必要时**：在混杂环境下，忽略因果结构的所有非因果方法均无法学习到有效策略（成功率≈0%）。
2. **因果调整不足以保证扩展性**：Causal BC和Causal GAIL在长时域任务中因复合误差或训练不稳定而性能严重退化。
3. **离策略Q学习与因果调整结合可扩展**：Causal SQIL和Causal IQ-Learn成功将因果模仿扩展至H≥1000的高维连续控制，性能超过先前CIL方法，有时甚至超越部分可观察的专家。
4. **窗口化近似有效**：假设混杂影响有界（k-有界），将调整集计算从O(H)降至O(k)，且中等k（1-10）最优。
5. **混杂偏置在训练分布内不可见**：图4显示训练时非因果方法与因果方法回报相近，但部署时崩溃，说明无法通过训练内验证诊断混杂。

- **方法创新与通用性**：首次将离策略软Q模仿学习（SQIL、IQ-Learn）与因果调整框架结合，且调整层与算法无关——未来任何基于软Q的IL算法都可套用该因果编码。
- **解决长时域可扩展性**：窗口化π-后门调整使因果模仿从仅有理论可行的短时域扩展到真实连续控制任务。
- **实验设计扎实**：同时对比多种因果/非因果方法，提供消融和敏感性分析，明确指出混杂只能在部署时暴露的实际现象。

- **可复现性**：开源代码仓库，且详细披露了超参数和网络结构（Appendix E）。

## 8. 不足与局限

- **依赖已知因果图**：实际中因果图需领域知识或数据估计，误设将导致调整集无效。文中承认是主要限制。
- **混杂影响有界假设**：假设因果关系在k步内衰减，适用于MuJoCo物理但可能不适用于长程依赖任务（如持续隐藏目标、慢变动力学），此时窗口化无效。
- **未利用奖励结构**：假设奖励完全隐藏，没有利用任何奖励先验。近期工作（Ruan et al., 2024）表明结合奖励先验可收紧下界甚至超越专家。
- **对专家次优性敏感**：在HumanoidMaze-Large（专家成功率仅7%）上，Causal SQIL和Causal IQ-Learn表现不如Causal BC，说明离策略Q学习在低质量专家下更容易被噪音侵蚀价值函数，而BC的监督克隆更稳健。
- **实验多样性有限**：仅测试两类移动机器人导航任务，缺乏障碍规避、操作等任务；未在真实机器人或视觉输入上验证（文中提到未来工作可组合扩散策略等）。
- **算力未报告**：影响复现和实用性评估。
