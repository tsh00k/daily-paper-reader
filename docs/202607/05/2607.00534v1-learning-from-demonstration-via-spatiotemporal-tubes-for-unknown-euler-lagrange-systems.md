## 摘要
我们提出了STT-LfD，这是一个统一的示教学习（LfD）框架，将运动学习与控制集成于未知的欧拉-拉格朗日系统。与追踪固定参考轨迹的传统解耦方法不同，所提出的方法将示教视为数据驱动的安全规范。利用异方差高斯过程，STT-LfD学习时空管道（STT）作为意图包络，捕捉任务随时间变化的精度要求。然后，一个闭式反馈控制器在尊重执行器限制的同时强制执行这些学习到的约束，无需显式系统辨识。该方法保留了示教的时间结构，计算效率高，且避免了显式系统辨识。在移动机器人和7自由度机械臂上的硬件实验表明，它在扰动鲁棒性和计算速度方面优于基线方法。

## Abstract
We present STT-LfD, a unified Learning from Demonstration (LfD) framework that integrates motion learning with control for unknown Euler-Lagrange systems. Unlike traditional decoupled approaches that track a fixed reference, the proposed method treats demonstrations as a data-driven safety specification. Using heteroscedastic Gaussian Processes, STT-LfD learns Spatiotemporal Tubes (STTs) as an intent envelope that capture time-varying precision requirements of a task. A closed-form feedback controller then enforces these learned constraints while respecting actuator limits, without requiring explicit system identification. The approach preserves the temporal structure of demonstrations, remains computationally efficient, and avoids explicit system identification. Hardware experiments on a mobile robot and a 7-DOF manipulator show that it outperforms baselines in robustness to disturbances and computational speed.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：从多次专家演示中学习**时空管道（Spatiotemporal Tube, STT）**，作为时变意图包络；设计闭式反馈控制器保证系统状态始终落在STT内，同时满足执行器约束。
- **关键技术细节**：
  - **数据预处理**：使用动态时间规整（DTW）对齐多次演示，消除人类执行速度差异带来的时间错位。
  - **STT学习**：对每个状态维度，采用**异方差高斯过程（HGP）** 建模：\(x_j(t) \sim \mathcal{GP}(\mu_j(t), \sigma_j^2(t))\)，其中\(\mu_j(t)\)和\(\sigma_j(t)\)分别为后验均值和标准差。STT定义为：\(\Gamma_j(t) = [\mu_j(t) - \lambda\sigma_j(t), \mu_j(t) + \lambda\sigma_j(t)]\)，\(\lambda\)为缩放参数（用户设定）。全维STT为笛卡尔积\(\Gamma(t) = \prod_{j=1}^n \Gamma_j(t)\)，形成时变轴对齐超矩形。
  - **控制器设计**：采用两步控制结构：
    - **速度级控制**：定义参考速度\(v_r(x,t) = -\text{diag}(\Psi(\varepsilon_x)) v\)，其中\(\varepsilon_x\)为归一化状态误差，\(\Psi\)为有界变换函数（如tanh），保证系统状态不离开STT。
    - **加速度级控制**：将速度误差约束在指数衰减漏斗\(\rho_v(t)\)内，最终控制输入\(\tau(t) = -\text{diag}(\Psi(\varepsilon_v)) \bar{\tau}\)，\(\bar{\tau}\)为最大允许扭矩。

  - **无需系统辨识**：控制器仅依赖有界变换函数和已知的执行器界限，不显式使用质量矩阵、科里奥利力等参数。
- **算法流程**（文字说明）：
