## 摘要
我们提出了排序然后行动（RTA），一个从专家视频演示中学习控制策略而无需环境奖励的框架。RTA离线训练一个视觉语言模型（VLM）作为基于进展的序数评分器，通过对打乱的帧序列使用组相对策略优化（GRPO）目标，迫使模型从视觉语义而非琐碎的时间线索中恢复时间顺序。重要的是，我们不是直接将评分器用作标量奖励模型，而是提出了一种基于相关性的奖励函数用于强化学习：在每个交互窗口中，我们计算预测进展排名与真实时间索引之间的斯皮尔曼秩相关，得到一个有界、尺度不变的学习信号。这种设计将奖励学习与绝对校准解耦，并能够在不同任务和环境之间稳定迁移。我们在离散控制基准（PyBoy: Catrap, Kirby）和连续控制任务（PointMaze, MetaWorld）上评估了RTA。RTA始终匹配或超越先前的基于视频的奖励学习方法和基于排名的基线，同时展示了单个预训练的进展评分器在跨任务重用中的强大能力。我们的结果表明，对视频衍生的序数信号应用相关性结构化监督足以进行策略学习，为显式奖励设计提供了一种可扩展的替代方案。

## Abstract
We introduce Rank-Then-Act (RTA), a framework for learning control policies from expert video demonstrations without environment rewards. RTA trains a Vision-Language Model (VLM) offline as a progress-based ordinal scorer, using a Group Relative Policy Optimization (GRPO) objective over shuffled frame sequences, which forces the model to recover temporal ordering from visual semantics rather than trivial time cues. Importantly, instead of using the scorer directly as a scalar reward model, we propose a correlation-based reward function for reinforcement learning: at each interaction window, we compute the Spearman rank correlation between predicted progress rankings and true temporal indices, yielding a bounded, scale-invariant learning signal. This design decouples reward learning from absolute calibration and enables stable transfer across tasks and environments. We evaluate RTA on discrete control benchmarks (PyBoy: Catrap, Kirby) and continuous control tasks (PointMaze, MetaWorld). RTA consistently matches or outperforms prior video-based reward learning methods and rank-based baselines, while demonstrating strong cross-task reuse of a single pretrained progress scorer. Our results suggest that correlation-structured supervision over video-derived ordinal signals is sufficient for policy learning, offering a scalable alternative to explicit reward design.

## 2. 论文提出的方法论：核心思想、关键技术细节

该论文提出了一个名为 **Rank-Then-Act (RTA)** 的两阶段框架，其核心思想是**利用视频帧之间的时序顺序（序数信息），而非具体的标量值，来构建学习信号**。这避免了绝对奖励校准的难题。

*   **核心思想**：
    1.  **序数而非标量**：不预测帧的“好坏”绝对值，而是预测其在整个任务过程中的相对“进展排名”。
    2.  **秩相关而非回归**：不要求奖励信号的绝对值准确，而是要求其与时间进度呈单调正相关。使用斯皮尔曼秩相关系数（Spearman rank correlation）来衡量这种相关性。
    3.  **打乱与锚定**：通过打乱帧序列并引入锚定帧（anchor），迫使模型学习基于视觉语义的任务进展，而非“越晚越好”的简单捷径。

*   **关键技术细节与流程**：
    *   **Stage 1: Rank (评分器训练)**
        1.  **输入准备**：从专家视频中提取帧序列，固定第一帧作为锚点（anchor），然后将其余帧随机打乱。
        2.  **模型与推理**：将打乱后的帧（包含锚点）送入一个视觉语言模型（VLM，如Qwen2.5-VL-7B）。模型输出对每个非锚点帧的“进展排名”预测值（一个标量）。
        3.  **优化目标**：使用 **GRPO (Group Relative Policy Optimization)** 来优化VLM。奖励函数定义为预测的进展排名向量（p）与真实时间索引向量（q）之间的**斯皮尔曼秩相关**: `R = spr(p, q)`。该奖励最大化时，鼓励模型输出的排名与时间顺序严格一致。
        4.  **输出**：一个训练好的、冻结的 **VLM进展评分器 (progress scorer)**。

    *   **Stage 2: Act (策略学习)**
        1.  **构建窗口**：在智能体与环境交互时，每N步（query step）构建一个长度为m的最近交互窗口 `W_t`，窗口内的第一帧作为锚点。
        2.  **计算奖励**：将窗口内的非锚点帧多次（L次）打乱，送入冻结的VLM评分器。每次得到一个进展排名预测，然后计算其与真实时间戳的斯皮尔曼秩相关。对所有L次的结果求平均，作为当前窗口的标量奖励 `r_t`。
        3.  **奖励特性**：该奖励值 `r_t` 在 [-1, 1] 之间，具有有界、尺度不变性。它衡量的是智能体行为在窗口内是否表现出**与任务进展一致的序贯性**。
        4.  **策略更新**：智能体（策略网络，如VL-DAC, PPO, DrQv2）仅使用这个 `r_t` 作为唯一的奖励信号进行策略梯度更新（如PPO），完全不需要环境的外部奖励。

## 6. 论文的主要结论与发现

1.  **序数监督的有效性**：通过在打乱的帧序列上训练VLM来预测进展排名，可以得到一个强大的、不受制于“越晚越好”短路的进展评分器。这使得从视觉语义中学习任务进展成为可能。
2.  **相关性奖励的优越性**：使用斯皮尔曼秩相关作为奖励信号，比直接使用VLM输出的标量值效果更好。这种有界、尺度不变的奖励信号为策略学习提供了稳定且信息丰富的反馈，避免了绝对校准问题。
3.  **无奖励策略学习**：RTA方法成功地在Catrap、Kirby、PointMaze和MetaWorld等多个离散和连续控制任务上学习到了有效策略，**完全不需要环境提供的任何外部奖励**。在Catrap的Level 2上达到了100%的成功率，并在Kirby上实现了非零的成功率，是唯一一个在该任务上取得进展的方法。
4.  **强大的泛化能力**：
    *   **跨关卡/任务**：在Catrap单个关卡上训练的评分器可以泛化到其他关卡，而在全游戏/多游戏上训练的评分器泛化能力最强。
    *   **跨领域**：在一个视觉领域（如GameBoy游戏、MetaWorld）训练的评分器可以为零个或低成本的另一个领域（如COIN数据集、YouTube视频）的策略学习提供有效奖励信号。
5.  **设计的关键性**：消融实验表明，合适的窗口大小（避免过小导致对局部一致性的过度敏感，或过大导致的信号稀释）以及多次打乱（L）是获得稳定且有效奖励信号的关键。

## 7. 优点

*   **方法简洁优雅**：核心思想简单——利用序数信息，通过秩相关构建奖励，避免了复杂的对抗性训练或多任务学习，易于理解和实现。

*   **强泛化性**：单一预训练的VLM评分器能够在任务、关卡甚至视觉领域之间直接复用，极大地提升了方法的可扩展性和实用性，是其主要亮点之一。

## 8. 不足与局限

1.  **对专家数据质量的依赖**：方法依赖多样化的专家视频数据。如果专家数据覆盖不全或存在偏差，可能会导致评分器过拟合到特定风格或策略上，从而削弱泛化能力。
2.  **窗口规模与时间不一致性问题**：如作者在“讨论”和“循环轨迹实验”中所述，窗口大小是一个关键超参数，需要找到中间稳定区间。在非全局单调进展的任务中，奖励信号可能会产生误导。对于需要长时间回溯或非单调逻辑的任务（如迷宫探索需要先退后进），该方法可能面临挑战。
3.  **VLM推理成本与延迟**：尽管使用了LoRA和稀疏奖励（每N步），但VLM推理仍然是高计算成本和延迟的主要来源，限制了其在实时控制场景中的应用。
4.  **部分任务性能仍弱于“混合奖励”方法**：在MetaWorld的部分任务中，纯RTA奖励（Only Ranking）的性能弱于Rank2Reward加GAIL的混合方法。这暗示在某些需要大量探索的场景下，纯序数信号可能不足以引导出最优探索策略，需要结合其他探索奖励（如GAIL，即论文中提到的“RTA”版本）。
5.  **未处理高度非单调场景**：虽然分析了循环轨迹，但论文承认，对于包含明确子目标、允许暂时退步的行动序列的任务，其纯窗口方法可能不够鲁棒，未来需要引入层级或多尺度时序聚合。
