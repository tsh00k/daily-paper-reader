## 摘要
在人形运动控制中，模型预测控制（MPC）提供了基于物理的预测和约束处理能力，而强化学习（RL）则通过大规模仿真实现了鲁棒的全身技能。然而，在强化学习内部使用MPC通常需要耗时的构建问题或过多的训练开销，使得此类框架在实践中难以证明其合理性。本文研究用于人形行走与操作的高效训练时MPC引导方法，称为MPC-RL。我们引入了一种基于质心动力学的MPC奖励公式，在训练时利用MPC轨迹的引导。为了使其在大规模并行RL中实用，我们开发了π^nMPC，一种并行时域、无需构建的批处理GPU MPC求解器，它直接处理时变动力学，以避免高内存使用和预编译。通过多种对比研究和硬件验证，我们发现在行走与操作技能上，MPC-RL取得了优越的性能。代码库可在https://github.com/junhengl/mpc-rl获取。

## Abstract
In humanoid motion control, model predictive control (MPC) offers physically grounded prediction and constraint handling, while reinforcement learning (RL) enables robust whole-body skills through large-scale simulation. However, using MPC inside RL often requires time-consuming problem construction or excessive training overhead, making such frameworks difficult to justify in practice. This work studies efficient training-time MPC guidance for humanoid locomotion and manipulation, termed MPC-RL. We introduce a centroidal-dynamics MPC reward formulation that leverages guidance from MPC trajectories in training time. To make this practical in massively parallel RL, we develop $π^n$MPC, a parallel-in-horizon and construction-free batched GPU MPC solver that operates directly on time-varying dynamics to avoid high memory usage and pre-compilation. Through a variety of comparative studies and hardware validations, we have found that MPC-RL achieves superior performance in locomotion and manipulation skills. The code base is available at https://github.com/junhengl/mpc-rl.

## 1. 核心问题与整体含义（研究动机和背景）
人形机器人运动控制面临两大范式：模型预测控制（MPC）提供基于物理的预测和约束处理能力，强化学习（RL）通过大规模仿真实现鲁棒的全身技能。然而，将MPC与RL结合通常面临两个主要挑战：一是训练时反复求解MPC会导致高昂的计算开销；二是现有MPC求解器在批处理时需要复杂的预编译或高内存消耗，难以在并行RL训练中实用。本文旨在研究一种高效、可扩展的训练时MPC引导方法（MPC-RL），同时将应用从行走扩展到操作任务，以实现更强大的人形机器人行为。

## 2. 方法论：核心思想、关键技术细节
### 2.1 整体架构
采用“训练时MPC引导 + 部署时无MPC”的框架。训练时，使用MPC为每个环境生成质心动力学预测轨迹，转换为“地标”奖励信号输入到PPO算法；部署时，学习到的策略直接运行，无需MPC参与。

### 2.2 质心动力学MPC（CD-MPC）
- **状态定义**：质心状态ξ = [c^T, l_G^T, k_G^T]^T（位置、线性动量、角动量）。
- **离散动力学**：基于质心动力学（CD）模型，用时间步Δt迭代更新位置、线性动量和角动量。
- **优化问题**：在有限时域N步内最小化阶段与终端成本，约束包括接触力锥（CWC）和摩擦锥，以及初始状态来自当前仿真。

### 2.3 MPC地标引导奖励（Landmark Guidance Reward）
- **地标提取**：从MPC预测轨迹中选取N_L个时间点（等距或自定义），生成预测地标。
- **预测置信度加权**：采用时域递减权重α（如{0.5, 0.25, 0.15, 0.07, 0.03}），对近端地标赋予更高权重，远端权重降低。
- **奖励函数**：
  r_mpc,c_t = exp( -∑ℓ αℓ ||e^ℓ_c(t)||^2 / σ_c^2 )
  其中e^ℓ_c(t) = c_t - ˆc_{t-δ_ℓ}(δ_ℓ)（当前CoM与之前MPC预测的该时刻CoM的误差）。同时有MPC线速度、角动量、足部位置、地面反力等奖励项。
- **总奖励**：r_t = r_reg_t + η r_mpc_t。

### 2.4 π^nMPC：并行时域、无构建的批处理GPU MPC求解器
- **基础**：基于πMPC的并行ADMM结构，但扩展至时变动力学的腿式机器人。
- **速度形式MPC**：优化输入增量Δu，将状态提升至[ξ; u_{k-1}]，有助于简化更新。
- **变量分裂方案**：引入辅助变量z_{k+1}=ξ̄_{k+1}和v_k=B̄_kΔu_k，将原问题分解为三个并行ADMM更新块：{Δu, ξ̄}、{z, v}、{θ, β, λ}。
- **关键特征**：所有ADMM子步在时域和环境中完全并行，预计算J_k、H_k等矩阵，避免编译和稀疏QP装配。
- **加速**：采用Nesterov加速与重启选项。

## 6. 主要结论与发现
- **MPC-RL优于纯RL**：在所有测试方向的速度跟踪、推拿恢复能力均有明显提升。
- **预测置信度加权（tapered）优于其他加权方案**：且优于CLF-RL和OGMP方法，训练收敛更快。
- **MPC更新率10 Hz和时域N=10（一个步长）为最佳设置**：过高频率或过长时域收益递减。
- **π^nMPC**：在批处理环境下（4096环境、长时域N=400）仍可求解，而其他求解器（qpth、qpax）内存不足或编译时间过长（CusADi需1.5小时编译），显著降低实用门槛。
- **操作任务**：MPC-RL引导使推盒子速度追踪更强，硬件上能推动290 kg推车，而纯RL训练会陷入停滞。

## 7. 优点
- **方法创新**：提出“预测置信度加权”地标奖励，整合了多时域预测信息；π^nMPC求解器实现并行时域和无编译操作，大幅降低训练开销。
- **实验扎实**：大量消融实验覆盖关键设计维度，包含仿真和硬件验证，对比多种Sota基线（CLF-RL、OGMP等）。

- **扩展性强**：从行走扩展到操作任务，证明框架通用性。

## 8. 不足与局限
- **模型简化**：仅采用简化质心动力学和预设接触时序，无法处理复杂地形下自主足端选择或全身动态交互（如攀爬）。
