## 摘要
可视化世界模型在学习复杂系统动力学方面展现出巨大潜力。最近的进展将这些模型用作模型预测控制（MPC）框架中的转移函数，以解决各类控制任务。然而，当应用于机器人时，它们仅限于单阶段任务，如抓取或拾取，难以应对需要复杂序列规划的多阶段任务。在这项工作中，我们提出了WorldDP，一种专为多阶段机器人操作设计的世界模型框架。我们的层次化方法利用高层世界模型作为转移函数，在运行时优化可行的子目标，然后由低层扩散策略实现这些子目标。为了进一步辅助学习动力学和规划，我们引入了以对象为中心的表示，将环境实体解耦，并使我们能够针对每个实体进行顺序规划。在多个机器人基准测试中，WorldDP始终优于现有基线，验证了将世界模型的物理基础规划与扩散策略的高效执行相结合能够获得优越的多阶段性能。

## Abstract
Visual world models have shown great potential in learning complex system dynamics. Recent advancements leverage these models as transition functions within Model Predictive Control (MPC) frameworks to solve various control tasks. When applied to robotics, however, they are limited to single-stage tasks such as reaching or grasping, and struggle with multi-stage ones that demand complex sequential planning. In this work, we introduce WorldDP, a world model framework designed for multi-stage robotic manipulation. Our hierarchical approach utilizes a high-level world model as a transition function to optimize for feasible subgoals during runtime, which are subsequently reached by a low-level Diffusion Policy. To further aid in learning dynamics and planning, we incorporate object-centric representations that decouple environmental entities and enable us to plan sequentially with respect to each. Evaluated across several robotics benchmarks, WorldDP consistently outperforms existing baselines, validating that coupling the world model's physically grounded planning with diffusion policy's efficient execution yields superior multi-stage performance.

## 2. 论文提出的方法论

### 核心思想：层次化框架 WorldDP

- 高层：**以对象为中心的世界模型**作为转移函数，在运行时通过MPC优化生成可行的**子目标**（subgoals）。
- 低层：**目标条件扩散策略（Diffusion Policy, DP）** 依次跟踪并执行这些子目标，将长时任务分解为短周期控制。
- 关键创新：对象中心表示解耦环境实体（机器人、物体、背景），允许对每个实体进行顺序规划；引入接触预测器和粒子滤波器优化，提升子目标生成质量。

### 关键技术细节

1. **对象中心编码器（Object-Centric Encoder, OCE）**：
   - 使用冻结的DINOv2提取图像块特征，通过Slot Attention框架将N个槽（slots）初始化为实体表示。
   - 槽矫正器（Slot Corrector）通过交叉注意力与GRU迭代精炼槽向量。
   - 训练时使用重建损失（L2距离）和基于SAM2分割掩码的Tversky损失，增强对小物体的关注。

2. **条件扩散Transformer动力学预测模型（CDiT）**：
   - 输入为对象中心状态 `s_k ∈ R^{N×d}` 和动作序列，动作通过Transformer编码为32维潜动作。
   - CDiT利用自适应层归一化进行条件建模，预测未来状态。
   - 训练时随机采样5帧（跨度≤100帧），通过教师强制预测，损失为MSE；推理时自回归滚动。

3. **任务执行（层次化MPC）**：
   - **粒子滤波器优化**：初始化Q条动作粒子（基于空间轨迹种子），执行自回归滚动评估代价，选取前M个精英作为下一轮均值，迭代L次，获得最优子目标序列。
   - **对象中心代价函数**：仅对目标物体嵌入计算MSE；同时引入**接触预测器**（MLP），输出各物体接触概率，使用交叉熵损失惩罚与期望接触模式的偏差。
   - **层次化执行**：依次对一个物体规划并执行所有子目标，捕获新观测，再规划下一物体，维持MPC循环。

4. **低层扩散策略（DP）**：
   - 基于Transformer的DP，输入为当前和子目标的对象中心状态扁平向量，结合末端执行器位姿与速度，输出40步动作序列。
   - 采用随机目标采样策略增强时序泛化；末端夹爪使用深度相机进行残差校正。

### 公式与算法流程（文字说明）

- 训练损失：`L_oce = L2 + λ·Tversky`；动力学模型训练：MSE。
- 规划代价：`C = C_dist + λ_plan·C_ce`。
- 算法伪代码：输入观测、目标、动作均值，经粒子滤波器迭代优化，得到最优子目标，再由DP依次执行。

## 3. 实验设计

- WorldDP在所有多阶段任务中**一致优于**现有基线，尤其在Cube-Triple（3立方体成功率30%，第二名为12%）和Scene-Single-Composite（完整任务20%，第二名14%）。
