## 摘要
除了预测机器人动作，世界行动模型（WAMs）还能够生成未来的视觉观察。我们基于这种生成能力提出了循环生成重放（REGEN），一种持续模仿学习框架，它合成伪重放轨迹，使机器人策略能够在没有存储原始人类演示的情况下练习先前学过的任务。在持续适应过程中，REGEN递归地查询WAM，仅基于先前的任务指令和当前任务观察来合成伪重放轨迹。在仿真和真实世界操作设置中的实验表明，相对于顺序微调，REGEN将灾难性遗忘减少了高达50%，同时接近需要访问真实重放数据的特权经验重放方法的性能。最后，我们分析了限制生成重放的因素，识别出长视界视觉退化以及动作-观察不一致是主要瓶颈。我们的结果确立了WAMs作为无需存储演示的持续机器人学习的有前途的基础。

## Abstract
Going beyond predicting robot actions, World Action Models (WAMs) can also generate future visual observations. We build on this generative capability to propose Recurrent Generative Replay (REGEN), a continual imitation learning framework that synthesizes pseudo-replay trajectories, enabling a robot policy to rehearse previously learned tasks without storing their original human demonstrations. During continual adaptation, REGEN recursively queries the WAM to synthesize pseudo-replay trajectories conditioned only on prior task instructions and current-task observations. Experiments in both simulation and real-world manipulation settings show that REGEN reduces catastrophic forgetting by up to $50\%$ relative to sequential fine-tuning, while approaching the performance of privileged experience replay methods that require access to real replay data. Finally, we analyze the factors limiting generated replay, identifying long-horizon visual degradation and action-observation inconsistency as the primary bottlenecks. Our results establish WAMs as a promising foundation for continual robot learning without stored demonstrations.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：在机器人持续模仿学习中，当顺序微调（Sequential Fine-Tuning）策略以学习新任务时，会遭遇**灾难性遗忘（Catastrophic Forgetting）**——即先前学到的技能迅速丢失。传统缓解方法（如经验重放 Experience Replay）需要存储原始人类演示，这面临存储成本高、隐私风险大以及现代专有数据集不可公开获取等现实制约。因此，亟需一种**无需存储真实旧演示**的持续学习机制。
- **研究动机**：世界行动模型（World Action Models, WAMs）不仅能预测动作，还能生成未来的视觉观察。这种生成能力为构建“原生重放机制”提供了独特机会——即让WAM自身模拟先前任务的轨迹，从而在不依赖真实演示的情况下进行排练。
- **整体含义**：论文提出了 **REGEN（Recurrent Generative Replay）** 框架，首次将WAM的生成能力用于持续模仿学习，证明了“一个能想象自己过去的机器人可以持续学习其未来”。该工作为无存储演示的终身机器人学习开辟了新的方向。

## 2. 方法论：核心思想、关键技术细节与算法流程

- **核心思想**：利用WAM同时预测动作与未来观测的能力，在持续适应新任务时，**递归地生成**先前任务的伪轨迹，并将这些伪轨迹与新任务演示混合训练，从而在不存储任何真实旧演示的情况下缓解遗忘。
- **关键技术细节**：
    1. **初始化阶段（0 ≤ t < H）**：从当前任务演示中取真实观测作为输入，因为递归生成需要至少一个真实观测上下文。
    2. **循环生成阶段（H ≤ t ≤ Tmax）**：不再使用真实观测，而是将模型自身生成的未来观测作为下一时间步的输入，递归进行：
       ```
       o_in_t = { o_t (t < H);  ˜o_t (t ≥ H) } 
       (˜a_{t:t+H}, ˜o_{t+H}, ˜r_t) ~ π_θ(· | o_in_t, ℓ_i)
       ```
    3. **终止条件**：当预测的目标奖励（task progress）连续3步 > 0.99 且至少一次达到1.0时停止；否则达到最大步长 Tmax。
    4. **伪轨迹构建**：每一步将输入观测与预测动作块的首个动作配对，形成伪轨迹 ˜τ_i。对每个先前任务生成N个伪轨迹，构成重放集 R_k。
    5. **训练目标**：使用联合数据集 D_k ∪ R_k 进行行为克隆，其中D_k为当前任务真实演示，R_k为生成的伪轨迹。损失函数为行为克隆损失，不改变WAM的其他生成目标。
- **算法流程**（文字描述）：
    - 对于每个先前任务 T_i：
        - 循环N次（每次生成一条伪轨迹）：
            - 用当前任务真实观测初始化前H步；
            - 从H步开始递归：将上一步生成的观测喂入WAM，得到新动作块和新观测；
            - 根据奖励预测判断是否终止；
            - 收集观测-动作对组成伪轨迹。
    - 将伪轨迹与当前任务演示合并，更新WAM参数。

## 3. 实验设计：数据集/场景、Benchmark、对比方法

- **仿真环境**：**LIBERO 基准**[13]，包含三个任务套件：
    - LIBERO-Spatial：空间泛化（物体布局变化）
    - LIBERO-Object：物体泛化（不同物体放入篮子）
    - LIBERO-Goal：目标泛化（不同目标与操作）
    - 每套件10个任务，每任务50条高质量遥操作演示。持续学习设置：6个任务作为基阶段，剩余4个任务依次引入（单任务顺序学习）。
- **真实世界环境**：xArm7 机器人单臂操作，三个任务（T1: 胡萝卜放入碗；T2: 胡萝卜放在盘子；T3: 茄子放入碗），每任务50条演示，顺序引入 T1→T2→T3。
- **基准方法**：
    - **Seq-FT**（顺序微调）：无缓解遗忘的基线。
    - **Seq-LoRA**：使用LoRA的低参数微调。
    - **EWC**（弹性权重巩固）：正则化方法。
    - **PackNet**：迭代剪枝分配任务参数。
    - **ER**（经验重放）：存储真实演示混合训练，作为上界参考（违背无真实数据假设）。
    - **RAR**（Rollouts-as-Replay）：在仿真器中回滚先前任务的策略收集轨迹，隔离生成重放与仿真器重放。
    - **REGEN（ours）**：仅使用WAM生成伪轨迹。
- **额外对比**：将WAM (Cosmos-Policy) 与 VLA (π0.5) 在LIBERO-Goal上进行持续学习对比。

## 6. 主要结论与发现

1. **REGEN显著减轻遗忘**：在所有LIBERO套件中，相比Seq-FT，NBT（负向向后迁移）降低超过50%（如LIBERO-Object从82.6降至26.1，LIBERO-Goal从100降至44.9）。AUC提升明显（如LIBERO-Object从24.9升至65.5）。
2. **接近特权ER的性能**：虽然在大多数设置中仍低于ER（尤其是LIBERO-Goal上差距较大），但REGEN在不存储任何真实演示的条件下达到了与ER相近的保留率，体现了WAM生成能力的有效性。
3. **生成质量是主要瓶颈**：伪轨迹的PSNR随持续阶段单调下降，且与实际执行成功率之间存在巨大脱节（想象成功率83%但实际仅42%）。长时域视觉退化与动作-观察不一致是限制生成重放性能的核心因素。
4. **WAM优于VLA**：与π0.5相比，WAM由于具备联合生成观测的能力，使得REGEN在持续学习上比仅预测动作的VLA（即使有更大预训练）取得更好保留。
5. **表示漂移小**：REGEN的动作隐空间中心偏移（0.12）远小于Seq-FT（0.30），接近ER（0.04），表明生成重放有助于保持策略内部表示稳定。

## 7. 优点

- **方法创新**：首次将WAM的生成能力直接用于持续学习中的重放，实现了无存储演示的rehearsal机制，方向新颖且实用价值高。
- **实验设计全面**：涵盖仿真与真实环境、多种任务分布、多种对比基线（包括上界ER），并深入分析生成瓶颈，验证了方法的有效性和局限性。
- **分析深入**：通过PSNR、表示漂移、想象-实际成功率不一致等多维度分析，揭示了当前WAM生成质量的限制，为后续改进提供了清晰方向。
- **可复现性**：代码和伪代码均已提供（Appendix A.2），算法细节清晰，便于复现。

## 8. 不足与局限

- **实验覆盖有限**：
    - 仅测试了单一WAM架构（Cosmos-Policy），未评估其他WAM（如GigaWorld-Policy、MotoBrain）对REGEN的影响。
    - 真实世界任务数量少（3个）且均为桌面抓放操作，缺乏复杂长时域任务（如多步骤操作）的验证。
    - 持续学习任务序列长度短（LIBERO中4个任务，真实中3个），未探索更长序列下生成质量的累积退化效应。
- **性能差距**：与ER相比仍有明显差距（LIBERO-Goal上NBT：44.9 vs 7.2；AUC：40.8 vs 92.4），说明生成重放尚不能完全替代真实重放。
- **生成质量退化**：伪轨迹的视觉质量随阶段下降，且动作-观察不一致严重，导致重放信号质量递减。当前WAM难以在递归生成中保持长期一致性和高保真度。
- **依赖初始观测**：REGEN需要从当前任务的真实观测初始化生成，若当前任务环境与先前任务差异极大（如物体缺失），则生成可能不可靠（如LIBERO-Spatial需要特殊处理）。
- **计算开销**：每个持续阶段需为每个先前任务生成多条伪轨迹（文中使用10条），增加了训练时间开销（尽管推理时可并行化）。
- **未讨论灾难性遗忘的评估指标偏斜**：NBT基于成功率比值，对于成功率接近0的任务可能产生极端值，论文未进行鲁棒性分析。
