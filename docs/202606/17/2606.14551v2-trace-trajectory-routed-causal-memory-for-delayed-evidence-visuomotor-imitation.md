## 摘要
自主运行的机器人可能需要根据不再可见的证据做出决策。我们研究延迟证据任务，其中早期线索在后期决策点之前消失，因此视觉上相似的观察可能需要不同的行动。在这些设置中，当前观察不足以作为控制的状态。我们引入了轨迹路由因果证据（TRACE），一种用于视觉运动模仿策略的记忆框架。TRACE将任务相关的视觉和机器人状态证据（如物体身份、目标选择或路线相关状态）存储在固定大小的潜在记忆中，该记忆在长片段中保持有界。TRACE不是通过原始时间或手动提供的任务标签来索引记忆，而是使用路径签名：执行机器人状态轨迹的紧凑、顺序敏感特征。这些签名不存储视觉线索本身；相反，它们提供了轨迹条件的键，用于写入和检索当线索可见时存储的证据。当机器人后来遇到模糊观察时，策略依赖于TRACE记忆来恢复缺失的上下文并选择正确的分支。TRACE通过轻量级适配器附加到策略上，而不改变策略主干、动作头或模仿目标。在具有视觉模糊分支点的真实世界长视野操作任务中，TRACE相比替代基线（包括短历史和循环记忆）提高了分支选择和任务成功率。项目页面：https://jeong-zju.github.io/trace

## Abstract
Robots under autonomous operation may require decisions based on evidence that is no longer visible. We study delayed-evidence tasks, where an early cue disappears before a later decision point, so visually similar observations can require different actions. In these settings, the current observation is not a sufficient state for control. We introduce TRAjectory-routed Causal Evidence (TRACE), a memory framework for visuomotor imitation policies. TRACE stores task-relevant visual and robot-state evidence, such as object identity, target choice, or route-dependent state, in a fixed-size latent memory that remains bounded over long episodes. Instead of indexing memory by raw time or manually provided task labels, TRACE uses path signatures: compact, order-sensitive features of the executed robot-state trajectory. These signatures do not store the visual cue itself; rather, they provide trajectory-conditioned keys for writing and retrieving the evidence stored when the cue was visible. When the robot later reaches an ambiguous observation, the policy conditions on TRACE memory to recover the missing context and choose the correct branch. TRACE attaches through lightweight adapters to policies, without changing the policy backbone, action head, or imitation objective. Across real-world long-horizon manipulation tasks with visually ambiguous branch points, TRACE improves branch selection and task success over alternative baselines, including short-history and recurrent memory. ## 1. 论文的核心问题与整体含义（研究动机和背景）

- 核心问题：在机器人视觉运动模仿（visuomotor imitation）中，存在一类“延迟证据任务”（delayed-evidence tasks）：早期线索（如物体身份、起始位置）在后续决策点之前消失，导致视觉上相似的观察需要采取不同的动作。标准策略通常假设当前观测或短历史窗口足以进行控制，但在延迟证据情境下这一假设失效，因为当前观测无法区分不同历史分支。
- 整体含义：为解决此类部分可观测问题，需要一种**因果、固定预算的记忆机制**，在线更新并保留早期任务证据直到需要时使用。TRACE（Trajectory-routed Causal Evidence）正是为此设计的记忆框架，旨在提升机器人模仿策略在视觉模糊分支点下的决策能力。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 将任务相关的视觉和机器人状态证据存储在一个固定大小的潜在记忆槽（latent memory slots）中，并利用执行轨迹的**路径签名（path signature）**作为轨迹条件键来指导记忆的写入和读取。
- 记忆内容（视觉/状态证据）与记忆索引（轨迹签名）分离，使机器人能在当前观测不足以决策时，通过签名-路由恢复历史上下文。

### 关键技术细节
1. **路径签名（Path Signature）**：对机器人状态轨迹进行紧凑、有序敏感的摘要。给定分段线性轨迹 \( S_t: [0,1] \rightarrow \mathbb{R}^{d_s} \)，其深度 \( p \) 的路径签名包含各级积分项，对时间重参数化不变，但对轨迹顺序敏感。TRACE 使用流式签名 \( \xi_t \) 和差分签名 \( \delta_t \)。
2. **签名路由槽记忆**：
   - 轨迹键：\( q_t = \phi_q([g_t, \Delta g_t]) \)，其中 \( g_t = \phi_g(\xi_t) \)，\( \Delta g_t = \phi_\Delta(\delta_t) \)。
   - 路由：用轨迹键计算每个记忆槽的权重 \( \omega_{t,k} \)，通过 softmax 得到写分布。
   - 写入：采用门控更新，\( m_{t,k} = (1 - \beta_{t,k}) m_{t-1,k} + \beta_{t,k} \tilde{m}_{t,k} \)，其中 \( \beta_{t,k} \) 是路由调制的写入门。
   - 读取：基于当前证据和路由状态计算读取权重 \( \alpha_{t,k} \)，得到记忆读出 \( z_t^{\text{mem}} \)。
3. **策略适配器**：TRACE 通过轻量适配器将记忆状态注入基础策略的条件空间，不改变策略主干、动作头或模仿损失。对回归型策略（如ACT）使用记忆 token 拼接；对扩散策略使用全局条件向量。
4. **训练**：采用因果扫描，只使用当前及之前的观测和状态，不使用未来信息或线索标签。引入辅助损失（平衡、熵、一致性）防止槽坍缩、扩散写入和读写不匹配。

### 任务场景
- 五个真实世界长时操作任务（每个任务包含延迟证据分支）：

## 7. 优点：方法或实验设计上的亮点

- **模块化与即插即用**：TRACE 通过轻量适配器附加到任意视觉运动策略，不改变其主干、动作头或损失函数，实用性强。
