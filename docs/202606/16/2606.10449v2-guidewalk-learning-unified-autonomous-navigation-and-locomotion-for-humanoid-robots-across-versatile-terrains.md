## 摘要
人形机器人已具备强大的运动能力，但在多种地形上的可靠导航仍具挑战性，因为避障必须与动态可行的运动相协调。在这项工作中，我们提出了GuideWalk，一个统一的端到端框架，它将可穿越性感知的导航引导与地形自适应运动教师相结合，用于人形机器人导航。具体而言，我们引入了一个导航模块，提供明确的速度引导，将避障与地形条件解耦，从而在多样环境中实现稳健规划。我们提出了一种复合教师蒸馏方案，其中目标导向命令和动态一致的动作被聚合并蒸馏为单一策略。为进一步提升鲁棒性，蒸馏策略通过强化学习和辅助行为克隆目标进行优化，该目标在保留期望教师行为的同时促进探索。实验表明，GuideWalk在保持稳定人形运动的前提下，实现了稳定且有效的导航。

## Abstract
Humanoid robots have achieved strong locomotion capabilities, but reliable navigation on versatile terrains remains challenging because obstacle avoidance must be coordinated with dynamically feasible motion. In this work, we present GuideWalk, a unified end-to-end framework that integrates traversability-aware navigation guidance with terrain-adaptive locomotion teacher for humanoid navigation. Specifically, we introduce a navigation module that provides explicit velocity guidance, decoupling obstacle avoidance from terrain conditions to enable robust planning across diverse environments. We propose a composite teacher distillation scheme, where goal-directed commands and dynamically consistent actions are aggregated and distilled into a single policy. To further improve robustness, the distilled policy is refined with reinforcement learning and an auxiliary behavior cloning objective, which promotes exploration while preserving desirable teacher behaviors. Experiments demonstrate that GuideWalk achieves stable and effective navigation while maintaining stable humanoid locomotion.

## 2. 方法论

### 核心思想
- 采用**复合教师蒸馏+两阶段训练**流程：第一阶段通过DAgger（Dataset Aggregation）从复合教师（导航模块+运动教师）蒸馏学生策略；第二阶段使用PPO强化学习并结合辅助行为克隆（BC）损失进行微调，以在探索中保留教师行为。

### 关键技术细节
- **导航模块**：基于动态窗口法（DWA）的局部规划器，评估候选速度指令（前向线速度v和偏航角速度ω），通过最大化加权目标函数选择最优速度。目标函数包括：航向对准度、障碍物距离、前进速度偏置。适应差分驱动力学模型以匹配运动教师的跟踪能力。公式：  
  `(v*, ω*) = argmax G(v,ω)=σ(α·heading+β·dist+γ·velocity)`, 满足速度约束集。
- **运动教师（Locomotion Teacher）**：采用带对抗运动先验（AMP）的RL训练，从自收集的动作捕捉（MoCap）数据中学习类人运动，输出全身关节动作。该教师利用特权观测（地面真实海拔图、无噪声本体感知等），实现鲁棒地形自适应运动。
- **复合教师**：导航模块输出速度指令，输入运动教师生成最终关节动作，形成确定性监督基线。
- **第一阶段：DAgger蒸馏**：学生策略基于部分可观测信息（深度图、海拔图、本体感知）与环境交互，收集学生状态分布下的轨迹，查询复合教师获得目标动作，迭代优化均方误差损失，使学生策略模仿教师行为。
- **第二阶段：RL微调**：使用PPO最大化累积回报（奖励设计包含速度跟踪、目标接近、姿态保持等），同时附加BC损失（学生与教师动作的MSE），联合优化：  
  `L = L_PPO + λ·L_BC`，λ=0.2。

### 算法流程（文字说明）
1. 预训练运动教师：在仿真中使用RL+AMP，基于特权观测和MoCap数据训练地形自适应运动策略。
2. 构建复合教师：将DWA导航模块与运动教师串联，作为蒸馏监督源。
3. Stage 1 - 蒸馏：学生策略在环境中滚动，收集状态，冻结复合教师提供目标动作，通过DAgger迭代更新学生策略参数。
4. Stage 2 - 微调：继续环境交互，使用PPO优化累积回报，同时计算在线BC损失，联合更新策略参数，直至收敛。

## 6. 主要结论与发现

- GuideWalk在多种地形上（楼梯、斜坡、窄梁、障碍物环境）实现了稳定有效的导航，同时保持人形机器人运动稳定性。
- **导航模块必要性**：移除导航模块后，障碍物场景导航成功率大幅下降（99.0% → 69.9%），轨迹出现振荡和无效绕行，说明显式速度引导对避障决策至关重要。
- **运动教师必要性**：移除运动教师导致复杂地形穿越成功率显著降低（如窄梁从99.8%降至0%），且产生剧烈上身运动导致频繁碰撞，说明运动先验对于生成动态可行动作不可或缺。
- **蒸馏阶段作用**：移除蒸馏（Stage 1）虽不影响最终性能，但大幅增加训练时间（RL每更新0.24s vs DAgger 0.08s）；蒸馏提供高效初始化。
- **RL微调作用**：仅蒸馏（Stage 2）导致导航性能退化（NSR仅18.3%），因缺乏碰撞惩罚等反馈；RL微调结合BC损失可发现新行为（如侧移、缩臂避障），显著提升鲁棒性。
- **联合效益**：两阶段协同作用，蒸馏提供强初始化和教师行为，RL通过环境交互进一步优化，最终策略超越单个教师表现。

## 7. 优点

- **端到端统一框架**：将导航与运动学习集成到单一策略中，避免了分层流水线的信息损失和复杂度，学习效率高。
- **导航-运动解耦与耦合的巧妙平衡**：通过DWA提供显式速度指令作为中间表示，既解耦了避障决策与地形适应，又为运动教师提供了可跟踪的平滑指导。
- **复合教师蒸馏**：结合导航模块和运动教师的互补监督，使学生策略同时获得避障策略和地形自适应运动技能，知识迁移高效。
- **两阶段训练**：Stage 1快速初始化，Stage 2通过RL+BC提升鲁棒性和探索，有效防止灾难性遗忘，训练效率优于纯RL。
- **感知丰富**：同时利用深度图像和海拔图，分别处理障碍物和地形几何，提高对环境理解的能力。
- **真实世界验证**：在真实Kuavo机器人上成功部署，展示了从仿真到现实的迁移能力。

## 8. 不足与局限

- **硬件冗余与计算开销**：同时依赖LiDAR（构建海拔图）和深度相机，增加了传感器成本和计算负担。论文自承未来将探索从深度序列隐式重建海拔图，以消除LiDAR需求。
- **局部规划局限**：DWA作为局部导航器，在死胡同、长距离迂回等需要全局路径规划的复杂场景中可能失败。论文承认目前缺乏全局规划（如A*、RRT*）集成。
- **实验对比不足**：缺乏与现有端到端或多层导航方法的定量对比（如VB-Com、TNavRL、FocusNav等），仅提供自身消融，无法证明相对于其他方法的优势。
- **真实世界实验局限于定性**：仅展示3个简单场景的视频定性分析，未报告真实世界定量指标（成功率、时间、碰撞次数等），泛化性难以评估。
- **仿真场景泛化性有限**：虽然涵盖多种地形，但障碍物布局和地形复杂度可能仍与现实差距，且在动态障碍物、非平坦非结构化地面等场景未测试。
- **可重复性风险**：依赖自收集MoCap数据和特定仿真环境（Isaac Sim），未开源代码或模型，可能限制结果复现和社区应用。
